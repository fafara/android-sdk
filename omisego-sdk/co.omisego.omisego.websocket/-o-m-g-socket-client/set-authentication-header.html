setAuthenticationHeader

fun setAuthenticationHeader(apiKey: String, authenticationToken: String): Unit

Set new authentication header. This will send leave request for every channel to the server. You'll need to join the channel again (please wait for disconnected listener is invoked).

Parameters

apiKey - is the API key (typically generated on the admin panel).

authenticationToken - is the token corresponding to an OmiseGO Wallet user retrievable using one of our server-side SDKs.
