OMGSocketClient

class OMGSocketClient : Client, SocketClient, SocketConnectionListenerSet, SocketChannelListenerSet, SocketCustomEventListenerSet

The OMGSocketClient represents an object that knows how to interact with the eWallet Web Socket API. An instance should be created by using OMGSocketClient.Builder. For example,

    val omgSocketClient = OMGSocketClient.Builder{
         clientConfiguration = config
         debug = false
    }.build()

The available methods and details are listed in the SocketClientContract.Client
