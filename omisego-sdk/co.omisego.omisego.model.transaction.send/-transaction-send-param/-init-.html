TransactionSendParam(from: String?, to: String, amount: Double, tokenId: String, metadata: Map<String, Any>, encryptedMetadata: Map<String, Any>)

Represents a structure used to create a transaction

Parameters

from - The address from which to take the tokens (which must belong to the user). If not specified, the user's primary address will be used.

to - The address where to send the tokens.

amount - The amount of token to transfer (down to subunit to unit).

tokenId - The id of the token to send.

metadata - Additional metadata for the transaction.

encryptedMetadata - Additional encrypted metadata for the transaction.
