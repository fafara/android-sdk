approveTransactionConsumption

fun approveTransactionConsumption(request: TransactionConsumptionActionParams): OMGCall<TransactionConsumption>

Asynchronously approve the transaction consumption from the given TransactionConsumptionActionParams object if success the OMGCallback will be invoked with the co.omisego.omisego.model.transaction.consumption.TransactionConsumption parameter, if fail the OMGCallback will be invoked with the co.omisego.omisego.model.APIError parameter.

Parameters

request - The TransactionConsumptionActionParams object containing the transaction consumption id to be approved.
