getSettings

fun getSettings(): OMGCall<Setting>

Asynchronously send the request to get the global settings of the provider. The global settings will contain a list of an available tokens to be used.

if success the OMGCallback callback will be invoked with Setting parameter, if fail OMGCallback will be invoked with the co.omisego.omisego.model.APIError parameter.
