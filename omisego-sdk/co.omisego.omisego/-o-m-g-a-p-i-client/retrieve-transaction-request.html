retrieveTransactionRequest

fun retrieveTransactionRequest(request: TransactionRequestParams): OMGCall<TransactionRequest>

Asynchronously retrieve a transaction request from its formattedId if success the OMGCallback will be invoked with the co.omisego.omisego.model.transaction.request.TransactionRequest parameter, if fail the OMGCallback will be invoked with the co.omisego.omisego.model.APIError parameter.

Parameters

request - The formattedId of the TransactionRequest to be retrieved
