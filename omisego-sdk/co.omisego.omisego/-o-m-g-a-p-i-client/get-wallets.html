getWallets

fun getWallets(): OMGCall<WalletList>

Asynchronously send the request to retrieve wallets of a user corresponding to the provided authentication token. if success the OMGCallback will be invoked with the list of Wallet parameter, if fail OMGCallback will be invoked with the co.omisego.omisego.model.APIError parameter.
