getCurrentUser

fun getCurrentUser(): OMGCall<User>

Asynchronously send the request to transform the User corresponding to the provided authentication token. if success the OMGCallback will be invoked with the User parameter, if fail OMGCallback will be invoked with the co.omisego.omisego.model.APIError parameter.
