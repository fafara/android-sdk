class EWalletClient

The class EWalletClient represents an object that knows how to interact with OmiseGO API.

Create instances using EWalletClient.Builder and pass your implementation of OMGCallback interface to generate an implementation

For example, val config = ClientConfiguration( baseURL = "YOUR_BASE_URL", apiKey = "YOUR_API_KEY", authenticationToken = "YOUR_AUTH_TOKEN" ) val eWalletClient = EWalletClient.Builder { clientConfiguration = config debug = false }.build()
