TransactionCreateParams(fromAddress: String? = null, toAddress: String, toProviderUserId: String? = null, toAccountId: String? = null, amount: BigDecimal, tokenId: String, idempotencyToken: String = "$toAddress-${System.nanoTime()}", metadata: Map<String, Any> = mapOf(), encryptedMetadata: Map<String, Any> = mapOf())

Represents a structure used to create a transaction

Parameters

fromAddress - The address from which to take the tokens (which must belong to the user). If not specified, the user's primary address will be used.

toAddress - The address where to send the tokens.

toProviderUserId - The provider user id where to send the token.

toAccountId - The account id where to send the tokens.

amount - The amount of token to transfer (down to subunit to unit).

tokenId - The id of the token to send.

idempotencyToken - The idempotency token to use for send the transaction.

metadata - Additional metadata for the transaction.

encryptedMetadata - Additional encrypted metadata for the transaction.
