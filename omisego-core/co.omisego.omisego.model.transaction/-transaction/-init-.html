Represents a transaction.

Parameters

id - The unique identifier of the transaction.

status - The status of the transaction (pending, confirmed or failed).

from - The source representing the source of the funds.

to - The source representing the destination of the funds.

exchange - Contains info of the exchange made during the transaction (if any).

metadata - Additional metadata for the consumption.

encryptedMetadata - Additional encrypted metadata for the consumption.

createdAt - The creation date of the transaction.

error - An APIError object if the transaction encountered an error
