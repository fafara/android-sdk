A convenient function to build a collection of Filter object with the Filterable scope.

For example, val filterList = buildFilterList<Filterable.TransactionFields> { field -> // Test number comparator add(field.fromAmount eq 100) add(field.fromAmount neq 101) add(field.toAmount gt 1.3) add(field.toAmount gte 2.8) add(field.toAmount lt 10) add(field.toAmount lte 1.0)

    // Test string comparator
    add(field.status eq Paginable.Transaction.TransactionStatus.CONFIRMED)
    add(field.type eq "some_type")
    add(field.id startsWith "123")
    add(field.status contains "confirm")

    // Test boolean comparator
    add("success" eq true)
    add("fail" neq true)

}
