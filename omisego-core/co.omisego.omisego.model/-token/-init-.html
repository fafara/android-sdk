Token(id: String, symbol: String, name: String, subunitToUnit: BigDecimal, createdAt: Date, updatedAt: Date, metadata: Map<String, Any>, encryptedMetadata: Map<String, Any>)

Represents a token

Parameters

id - The id of the token

symbol - The symbol of the token

name - The full name of the token

subunitToUnit - The multiplier representing the value of 1 token, i.e: If I want to give or receive 13 tokens and the subunitToUnit is 1000, then the amount will be 13*1000 = 13000

createdAt - The creation date of the token.

updatedAt - The date when the token was last updated.

metadata - Any additional metadata that need to be stored as a HashMap.

encryptedMetadata - Any additional encrypted metadata that need to be stored as a dictionary.
